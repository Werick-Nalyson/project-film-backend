### Para executar o projeto siga os passos abaixo
</br>


1 . Instalar dependências:
```
  "yarn" ou "npm install"
```

2 . Rodar banco de dados:
```
  docker run -p 27018:27017 mongo
```

3 . Executar projeto (caso yarn/npm):
```
  "yarn start:dev" ou "npm run start:dev"
```

</br>
